# microsoft-ai-challenge-day
This is the repository for Microsoft AI challenge day 2nd

# How to run
## preparation
- install poetry
- set up vertual environment
```
poetry install
```
- create your own .env from .env.sample by checking Azure portal

## run web app
```
poetry shell
./run_app.sh
```

## run in console (another option)
```
python backend.py
```
